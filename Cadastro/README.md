# CALCULADORA DE ANÚNCIO
## DESCRIÇÃO DO PROJETO
Projeto para cadastro de cliente com nome de anúncio, datas de início e fim e valor investido.

# INSTALAÇÃO
```shell
npm install
```

# COMO EXECUTAR
Após realizar a instalçao executar o comando
```shell
node cadastro.js
```
